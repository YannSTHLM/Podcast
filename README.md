# Podcast

A simple static web app for browsing and playing podcast episodes.

Features
- Load RSS/JSON podcast feeds
- Play and download episodes
- Manage subscriptions
- OPML import/export (file and URL), with progress and import summary

Usage
1. Open `index.html` in a modern browser (works as a static site).
2. Use "Manage Subscriptions" to add/remove feeds or import an OPML file or OPML URL.

Development
- This is a static HTML/JS app; edit files and serve with a local static server for development.
## Install as a Web App (PWA)
This site supports being installed as a web app on most modern browsers.

- Desktop / Android (Chrome, Edge, etc.): use the browser's install prompt or click the **Install App** button in the site header or in the **Manage Subscriptions** modal. The browser will show a native install dialog.
- iOS (Safari): open the Share menu and choose **Add to Home Screen**; alternatively use the **Install App** button to see instructions.

After install the site runs in standalone mode (no browser UI) and the install button will be hidden.
